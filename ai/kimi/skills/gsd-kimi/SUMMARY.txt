🔧 INSTALLATION
═══════════════════════════════════════════════════════════════════════════

cd ~/.kimi/skills/gsd-kimi
./install.sh

Or manually:
    pip install -e ~/.kimi/skills/gsd-kimi


🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Initialize GSD project:
    cd your-project
    gsd init

2. Create milestone:
    gsd new-milestone

3. Work through phases:
    gsd discuss-phase 1    # Create context
    gsd plan-phase 1       # Create plans

4. Execute with kimi:
    kimi --agent-file ~/.kimi/skills/gsd-kimi/gsd-agent.yaml \
         --yolo \
         --prompt "Execute plans in .planning/phase-1/"

⚡ COMMANDS
═══════════════════════════════════════════════════════════════════════════

CLI Commands:
    gsd init                    Initialize GSD
    gsd status                  Show status
    gsd new-milestone           Create milestone
    gsd discuss-phase N         Create phase context
    gsd plan-phase N            Create phase plans

Kimi (with agent file):
    /gsd:init                   Initialize
    /gsd:status                 Show status
    /gsd:new-milestone          Create milestone
    /gsd:discuss-phase N        Discuss phase
    /gsd:plan-phase N           Plan phase
    /gsd:execute-phase N        Execute phase
